AI Real-Time Gym Trainer

A real-time AI-powered gym coaching application built using Python, Streamlit, MediaPipe, and SQLite.

Features
Real-time pose detection
Squat tracking
Push-up tracking
Bicep curl tracking
Shoulder press tracking
Workout progress dashboard
User authentication
SQLite database for workout history
Tech Stack
Python
Streamlit
MediaPipe
OpenCV
SQLite
Groq API
